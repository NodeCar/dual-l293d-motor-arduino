# dual-l293d-motor-arduino
An Arduino-based dual L293d motor module communicating through i2c protocol.

Hardware
--------
- Arduino motor control shield V1
- Arduino pro mini

## Pin-Pin map
Shield|Arduino
-------|------
GND|GND
RESET|RESET
DIR_LATCH|
DIR_EN|
DIR_CLK|
DIR_SER|
PWM0A|3
PWM0B|5
PWM1A|6
PWM1B|9
PWM2A|10
PWM2B|11
