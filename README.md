# dual-l293d-motor-arduino
An Arduino-based dual L293d motor module communicating through i2c protocol.

Hardware
--------
- Arduino motor control shield V1
- Arduino pro mini

## Pin-Pin map
Shield|Arduino
-------|------
GND|GND
RESET|RESET
DIR_LATCH|12
DIR_EN|7
DIR_CLK|4
DIR_SER|8
PWM0A|6
PWM0B|5
PWM1A|9
PWM1B|10
PWM2A|11
PWM2B|3
